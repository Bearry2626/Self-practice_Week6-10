ในคลาสมีเนื้อหาเรียนคือ
1. DOM
root element != root node 

parent = พ่อ/แม่ของ node
child = ลูกของ node
sibling = ลูกพี่ลูกน้องของ node
ancestor = node ใด ๆ ที่อยู่เหนือ node ปัจจุบัน
descendant = node ใด ๆ ที่อยู่ใต้ node ปัจจุบัน 

- document root node ลูกคนแรกคือ Doctype
- การ Enter มันจะนับเป็น text node 

**ควรจำ
มีคำว่า Element แสดงว่าเป็น element node
ไม่มีคำว่า Element แสดงว่าเป็น node ทั่วไป ไม่สนใจว่าเป็น element

get children มีได้ 2 แบบ
1. childNodes (all node type) ได้ลูกทั้งหมด
2. children (element type only) ได้ลูกที่เป็น element เท่านั้น

2. Nodes Types
มีทั้งหมด 12 อัน แต่ 5, 6, 12 deprecated ไปแล้ว

**ควรจำ
- เวลาเรียก nodeType มันจะเรียก value ออกมา แต่อ้างถึงตัวพิมพ์ใหญ่ได้ ดูจากตารางได้
- บางอันไม่มี NodeValue เป็น null เพราะบาง type ไม่มี
เช่น ถ้าเป็ฯ element type จะกลายเป็นชื่อ tag => Element.tagName
ถ้าเป็น type อื่น จะกลายเป็น # ตามด้วยชื่อ tag => "#document"
